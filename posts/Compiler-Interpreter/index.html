컴파일러

  • 한 번에 파일을 컴퓨터 언어로 번역함
  • 처음 실행에는 오래 걸리지만 번역하고 나면 파일로 만들어져 빠름
  • 전체 코드를 스캔 후 번역하기 때문에 실행 전 오류를 알 수 있음
  • 코드를 수정하면 다시 컴파일 해야함

컴파일러 언어 : C / C++ / JAVA

인터프리터

  • 실행 시 한 번에 한 줄씩 번역함
  • 한 줄씩 실행하기 때문에 컴파일러보다 느림
  • 실행 후 오류를 알 수 있음

인터프리터 언어 : JavaScript / Python / Ruby
